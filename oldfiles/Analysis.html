Methods

Experiment

Viable Vibrio fisheri cells were isolated from an exponentially growing culture using FlowCytometery. Populations were exposed to either a good environment (LBS), poor environment (minimal media), or poor environment with predation (minimal media with 133 Cafeteria/mL). The good environment populations were cultured in 200uL and initial population size increased geometrically from 1 to 2048. The other conditions were cultured in 75uL and initial population size ranged from 1 to 64 (semi-geometric increase). Populations were grown for a set period of time, after which the population was scored as established (1) or fail to establish (0) based on a minimum threshold for opitcal density or fluorescence.

Correction 2/10/15 Realized 96 well plates also had Stabb marine media, not LBS! Now the difference between good and poor is glycerol concentration, 40mM vs 20mM glycerol.

Model Fitting

The presence of Allee effects was tested based on Dennis, B. 2002. Allee effects in stochastic populations. Oikos 96: 389-401. Demographic stochasticity, which can produce Allee-like behavior, can be differentiated from true Allee dynamics when examining the probability of extinction/establishment as a function of population size. Populations subject to Allee effects have very different probabilities of extinction above and below the lower critial size. This dramatic change in rates results in an inflection point at the critical population size. Populations subject to demographic noise will lack this inflection point in the probability of extinction. The presence of an inflection point can then surve as a test for the presence of an Allee effect. We used the Weibull function that will be sigmoidal (ie. have an inflection point) when \(k>1\). \[ \begin{aligned} y &= 1-e^{(\frac{-x}{\lambda})^{k}} && \text{Expression for Weibull} \\ \ddot{x} &= \lambda \sqrt[k]{\frac{k-1}{k}} && \text{Inflection Point of Sigmoid} \end{aligned} \]

In this case, y is probability of establishment, and x is the initial population size.

The inflection point is bounded between (0, \(\lambda\)), therefor the threshold to detect an Allee effect should include \(k > 1\) and \(\lambda > 1\) cell/mL. Since different volume was used for the good environment, the \(\lambda\) threshold differs, 0.2 for the good environment and 0.075 for the poor environment. For ease of comparison, all treatments were converted to the same scale of cells/mL before parameter estimation. Population size was log transformed for model fitting.

Data Prep

Need to currate data files: -type -plate

Good Environment

The data for the good environment was collected at a different time/trials. Three trials of 96well plates were run for the good environment. The following code chunk starts with the OD by time then: 1. Combines all 3 trials 2. Adds treatment information for each well 3. Determines the time it took each well to reach the minimum detection limit 4. Determine the max value or population size for each well 5. Key populations as established/not based on max value 6. Calculate summary stats for the populations that established

The output of this process is saved in demo.sum. The populations were grouped by treatment to calculate summary statistics, this is saved in demo.stat.

Poor and Predation Environment: Trial 15

The data for the poor and predation environment is from Trial 15, which used a 384 well plate. The plate was read for OD and fluorescence. Fluorescence was read at 3 sensitivity levels. All levels lead to the same patterns. The following chunk reads in the combined OD and RFU by time then:

  1. Assigns read type 1 through 4 (type 1: OD, type 2-4: fluorsecence at different sensitivity)
  2. Adds treatment information for each well
  3. Determines the time it took each well to reach the minimum detection limit
  4. Determine the max value or population size for each well
  5. Key populations as established/not based on max value
  6. Calculate summary stats for the populations that established

The out put of this process is saved as final.sum. The populations were grouped by treatment to calculate summary statistic, this is saved in stat.
